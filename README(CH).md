## 介绍
1. **Emergency-Brakes** 可以检测摄像头服务是否被占用并报告  
2. **Emergency-Brakes** 可以快速关闭摄像头
3. **Emergency-Brakes** 可以防止摄像头打开，以保护隐私和安全。
