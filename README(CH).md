## 背景
如今，隐私泄露事件频繁发生，许多未受保护的输入设备都有很大的隐私泄露风险  
**Emergency-Brakes** 是一个用于软件隐私保护的轻量级应用程序  

## 介绍
**Emergency-Brakes** 可以检测 ``摄像头服务/麦克风服务/定位服务`` 是否被占用并报告  
除此之外， **Emergency-Brakes** 还可以快速停止这些服务，以防止私人信息泄露
